Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => mayDay/src/main/AndroidManifest.xml
* assets/ => mayDay/src/main/assets/
* libs/crashlytics.jar => mayDay/libs/crashlytics.jar
* libs/cup-v1.0.0.jar => mayDay/libs/cup-v1.0.0.jar
* lint.xml => mayDay/lint.xml
* res/ => mayDay/src/main/res/
* src/ => mayDay/src/main/java/
